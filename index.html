INTRODUCTION

A free agenda is a set of tools which allows an organization to publish open data about its events.

The currently running agendas are listed below:

  • Agenda Unica dei Movimenti Catanesi
  • Agenda Eventi del Bastione degli Infetti
  • Eventi condivisi con la cittadinanza - Lecce .

Do you have an organization which organize events and do you want to participate to the free agenda project with managing your own agenda? Please contact us by sending a mail to longo@dmi.unict.it. If the published events are with free entrance the required fee is minimal.

DATA

Events of each agenda are entered via a dedicated google form, stored in a google sheet and then made available with the following modes:

  1. downloadable OWL Ontology
  2. SPARQL endpoint,
  3. Internet Calendar,
  4. RSS Atom feed.

Visit each agenda (see the AGENDAS section) to get information about how to access its events data.
